# The Cube Game

This project uses Godot 4.6.1. You can download it [here](https://godotengine.org/download/archive/4.6.1-stable/) 

<details>

<summary>or follow the instructions to have it installed with command line tool</summary>

Install [dotnet](https://dotnet.microsoft.com/en-us/download) (you probably did already), then run 

```bash
dotnet tool install Chickensoft.GodotEnv --global
```

And now, inside the project directory, run

```bash
godotenv godot install
godotenv godot use
```

</details>

## Collaboration

`main` branch is protected for pushes. That means you cannot push there.

To collaborate, before you start your work, create a new branch. Call it whatever you want, for example `feature/boss-animation`.

Then, commit and push your changes into this branch. Create Pull Request from [here](https://github.com/xill47/the-cube-game/compare).

To make it somewhat easier, I suggest using using [Jujutsu VCS](https://docs.jj-vcs.dev/latest/install-and-setup/) and [GG](https://github.com/gulbanana/gg/releases) for its UI. That way, when working, you'll less likely to accidentally commit to main, and it's easier to shuffle and name/rename your changes.
